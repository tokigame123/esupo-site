# 合同会社えーすぽ Web Site

合同会社えーすぽの公式Webサイトです。

## Pages

- `index.html`: トップページ
- `news.html`: ニュース一覧
- `admin.html`: ニュース追加・編集・書き出しツール

## News

ニュースは `news-data.js` からトップページとニュース一覧へ自動表示されます。更新手順は `NEWS_EDITING_GUIDE.md` を参照してください。

## SEO

- ページ別タイトル・説明文
- canonical URL
- OGP / Xカード
- Organization / NewsArticle 構造化データ
- `robots.txt`
- `sitemap.xml`
